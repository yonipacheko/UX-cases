RBapp – Resurs Bank Mobile Redesign

Product Designer · UX Researcher · 2019

Problem

The mobile app had a clunky payment flow and a cluttered UI. Customer satisfaction was low (2.2/5 App Store rating).

💡 My Approach

  1. Identified Friction Points: Used user testing and reviews.
  2. Prioritized for Impact: Built a user-story map.
  3. Designed a Simplified Flow: Removed steps and boosted contrast.

📈 Results

  • +90% user satisfaction
  • App rating rose from 2.2 → 4.2
  • Faster payment flow & cleaner interface

✅ Summary

“It feels like a totally new app. So much easier to pay bills now.” – User via App Store

This redesign taught me that clarity and simplification often drive more value than adding features—especially when rooted in real user pain.
